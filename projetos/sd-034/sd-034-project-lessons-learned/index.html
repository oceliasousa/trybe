Olá! Meu nome é Océlia. Atualmente estou em Transição de carreira, cursando MBA em Data Science e Analytics na USP e estudando Desenvolvimento Web Full Stack no bootcamp da Trybe.

No início do meu curso de desenvolvimento web, mergulhei fundo nos princípios de HTML e CSS. Rapidamente, me familiarizei com a vastidão dos elementos HTML, desde parágrafos básicos até cabeçalhos complexos. A utilização de atributos enriqueceu ainda mais os detalhes dos conteúdos. Com o CSS, não só transformei designs simples, mas aprendi a estilizá-los de forma única. Conquistei o domínio do box model, manipulei tipografias e priorizei a acessibilidade.

LIÇÕES APRENDIDAS

  1. Entender os diferentes tipos de elementos e tags HTML, como parágrafos (p), cabeçalhos (h1 - h6), links (a), imagens (img), listas (ul, ol, li) e outros.
  2. Saber como usar atributos dentro de tags para modificar ou especificar características adicionais, como class, id, src, alt, href, etc.
  3. Compreender a importância do uso de elementos semânticos (header, footer, article, section, etc). para uma estrutura de página mais clara e acessível.
  4. Entender ARIA (Accessible Rich Internet Applications) e outros atributos para tornar o conteúdo web mais acessível a pessoas com deficiências.
  5. Usar seletores para aplicar estilos a elementos específicos e conhecer as diversas propriedades CSS para controlar layout, cores, fontes, etc.
  6. Conhecer propriedades como margin, padding, width e height para controlar o espaço ao redor e as dimensões dos elementos.
  7. Aprender a estilizar texto com propriedades como font-family, font-size, text-align, text-decoration, e outras.
  8. Entender o conceito do box model, incluindo margin, border, padding e content, e como eles afetam o layout da página.
  9. Compreender o uso de pseudo-classes (:hover, :active, :first-child, etc.) e pseudo-elementos (::before, ::after) para estilizar elementos específicos de uma página ou adicionar conteúdo decorativo.
  10. Personalizar listas ordenadas e não ordenadas usando propriedades como list-style-type e list-style-image.

PRÓXIMAS LIÇÕES A APRENDER

  • Como escrever instruções em JavaScript, usar comentários, e entender a estrutura básica do código.
  • Declaração de variáveis usando let, const e (anteriormente) var. Entendimento dos diferentes tipos de dados: string, number, boolean, object, null, e undefined.
  • Operadores aritméticos (+, -, *, /, %), de comparação (==, ===, !=, !==, etc) e lógicos (&&, ||, !).
  • Condições usando if, else if e else, bem como a estrutura switch.
  • Compreender e usar laços como for, while, e do...while, além de métodos como forEach para arrays.
  • Declaração de funções, parâmetros, retornos e escopos de função. Funções anônimas e funções de seta (arrow functions) também são frequentemente introduzidas.
  • Como declarar, acessar e manipular arrays. Uso de métodos de array como push, pop, shift, unshift, splice, e slice.
  • Declaração e manipulação de objetos, acesso a propriedades e métodos.
  • Como ouvir e responder a eventos do usuário, como cliques, entrada de teclado, e movimentos do mouse, usando addEventListener.
  • Selecionar, modificar, adicionar ou remover elementos HTML e alterar estilos CSS via JavaScript.

OUTROS CONTEÚDOS ENSINADOS NO CURSO

Front-end HTML, CSS e JavaScript, que são a base para o desenvolvimento web. Além disso, você também aprenderá sobre React, um popular framework JavaScript para construir interfaces de usuário.
Back-end Node.js, que é um ambiente de execução JavaScript para o servidor. Além disso, você aprenderá sobre Express.js, um framework web para Node.js. Você também será introduzido ao SQL para gerenciamento de bancos de dados, bem como MongoDB, um banco de dados NoSQL.
Metodologias Ágeis Metodologias ágeis de desenvolvimento de software, como Scrum e Kanban.
DevOps Práticas de DevOps, incluindo integração contínua e implantação contínua (CI/CD) usando ferramentas como o Jest.
Soft Skills Por fim, a Trybe também se concentra no desenvolvimento de habilidades interpessoais, como trabalho em equipe, comunicação eficaz e resolução de problemas.
